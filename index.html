Funktionsweise (VR-optimiert V2)

Diese Version nutzt ein Click-to-Select System, das perfekt für VR-Controller funktioniert.

Anleitung

  • Schritt 1: Klicke auf eine Aufgabe in der linken Liste (wird blau markiert)
  • Schritt 2: Klicke auf ein grünes "Frei"-Feld im Kalender
  • Fertig! Die Aufgabe wird automatisch eingeplant
  • Aufgabe verschieben: Klicke auf eingeplante Aufgabe → klicke auf neues Feld
  • Aufgabe löschen: Klicke auf das rote X
  • Auswahl abbrechen: Klicke erneut auf die ausgewählte Aufgabe

Regeln

  • Nur „Frei"-Felder (grün) können genutzt werden
  • Mehrstündige Aufgaben werden automatisch verteilt
  • Blockierte Felder (grau) sind für Unterricht reserviert
  • Die Bearbeitungszeit einer Aufgabe darf nicht aufgeteilt werden

How it works (VR-optimized V2)

This version uses a Click-to-Select system that works perfectly with VR controllers.

Instructions

  • Step 1: Click on a task in the left list (it will be highlighted blue)
  • Step 2: Click on a green "Free" field in the calendar
  • Done! The task is automatically scheduled
  • Move task: Click on a scheduled task → click on a new field
  • Delete task: Click on the red X
  • Cancel selection: Click again on the selected task

Rules

  • Only "Free" fields (green) can be used
  • Multi-hour tasks are automatically distributed
  • Blocked fields (gray) are reserved for classes
  • The processing time for a task may not be split up
